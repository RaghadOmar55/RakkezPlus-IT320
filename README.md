⏱️ Rakkez+ (Focus Management System)

Rakkez+ is a web-based productivity application designed specifically for students to help them build better study habits and manage their time effectively using the Pomodoro technique.


🌟 Key Features:

- Customizable Study Timer: Set, start, pause, resume, and end study sessions with adjustable durations.


- Smart Break System: Option to enable automatic 5-minute breaks every 25 minutes to maintain focus.


- Distraction Logging: Track interruptions by recording reasons for pausing sessions to analyze personal distraction patterns.


- Personalized Focus Tips: Receive tailored advice and tips based on your recorded study habits.


- User Profiles: Manage personal data, track progress, and receive notifications for new focus insights.



🛠️ Technology Stack:

- Frontend: HTML5, CSS3, JavaScript.

- Backend Support: Designed for PHP/MySQL integration.

- Development Tools: Apache NetBeans IDE, GitHub, Jira.
